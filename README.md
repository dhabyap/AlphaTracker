# AlphaTracker

Crypto alpha tracking dashboard — track tokens with Birdeye & API.

## Features
- Real-time token tracking via Birdeye API
- Screenshot analysis → auto-add tokens
- Portfolio monitoring dashboard

## Tech Stack
- HTML
- JavaScript
- FastAPI / Python backend
- MySQL / SQLite

## Getting Started
```bash
git clone https://github.com/dhabyap/AlphaTracker.git
cd AlphaTracker
pip install -r requirements.txt
python main.py
```
